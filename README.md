## 🌟 Sobre el Proyecto

Esta es una aplicación web dinámica de una sola página (**One-Page**) diseñada casi de forma cinematográfica para la marca personal de Deisy Mavares. El proyecto fue conceptualizado, estructurado y desarrollado con la poderosa asistencia de **Google AI Studio**, logrando una sinergia perfecta entre arquitectura de código limpio (Clean Code) y un diseño de autor ("Premium Espiritual").

El sitio web transmite exclusividad, confianza empresarial y bienestar holístico a través de una interfaz interactiva y altamente optimizada.

## 🚀 Tecnologías y Herramientas

Este frontend está construido con el siguiente "Modern Stack" y ecosistema ecosistema de librerías:

- **Core / Framework:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Superconjunto JS:** TypeScript
- **Framework CSS:** [Tailwind CSS](https://tailwindcss.com/)
- **Micro Interacciones:** [Framer Motion](https://www.framer.com/motion/) (para efectos avanzados de scroll y montajes físicos)
- **Sistema Gráfico:** [Lucide React](https://lucide.dev/)
- **Inteligencia Artificial:** Google AI Studio (Pair-programming y refinamiento UI/UX)

## ✨ Identidad Visual y UI/UX (Brand Identity)

Como punto central del diseño visual, el proyecto implementa un sistema gráfico "High-End Editorial":

- **Paleta Cromática Exclusiva:** Manejo del **Púrpura Profundo** (`#5C4386`) como peso corporativo, **Lavanda Suave** / Gris para los respiros visuales, y el toque de lujo en **Dorado Cálido**.
- **Glassmorphism y Radiant Glow:** Elementos de interface semi-translucidos (menú navegacional, redes sociales en el footer y tarjetas "hovered") soportados con leves relives de sombra morada difusa.
- **Tipografía Aristocrática:** Títulos orquestados en *Serif* con distintos pesos de grosor, complementados con textos de cuerpo *Sans-Serif* ultrafinos.

## 💻 Instrucciones para Servidor Local

Sigue estos pasos rápidos o clona este repositorio para probar el esquema en desarrollo (`dev`) dentro de tu máquina local:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/gjaen14/DeisyMavares.git
   cd DeisyMavares
   ```

2. **Instalar paquetes NPM:**
   Asegúrate de contar con Node.js en tu dispositivo.
   ```bash
   npm install
   ```

3. **Iniciar motor de render y servidor:**
   ```bash
   npm run dev
   ```
   *(El portal quedará disponible y sincronizándose comúnmente desde `http://localhost:5173/`)*

4. **Para la fase de Despliegue Público (Build):**
   ```bash
   npm run build
   ```
